Course Organization & Management Application

The Course Organization & Management Application is a web-based prototype designed to support students, instructors, and administrators in managing academic course information.

The application includes role-based dashboards, course content management, assignment management, student submissions, grading, feedback, announcements, lesson plans, learning objectives, user management, and administrative reports.

This project uses Flask, SQLite, HTML, CSS, GitHub, Jenkins, and pytest to demonstrate software engineering, DevOps, testing, and application lifecycle concepts.
