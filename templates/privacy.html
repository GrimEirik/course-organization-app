Application Privacy Statement

This Course Organization Application is a student-developed academic project created for COM-430 Software Engineering. The application is intended for demonstration, testing, and educational purposes only.

The system stores basic user information, course information, assignments, submissions, grades, feedback, lesson plans, learning objectives, and announcements in a local SQLite database.

No real student records, protected education records, financial information, medical information, or sensitive personal information should be entered into this prototype system.

In a production environment, additional protections would be required, including encrypted passwords, secure authentication, role-based authorization, audit logging, encrypted database storage, and compliance review.
